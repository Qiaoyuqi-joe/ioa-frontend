What is an Agent?

An agent is an autonomous entity in the IOA network that can perform specific tasks. It can be a specialized service, an LLM instance, a tool, or a computing resource.

Agent Types

  • Agent: Task-performing service
  • LLM: Language model instance
  • Tool: Utility service
  • Computing: Computing resource

Resource Allocation

Set the percentage of CPU and memory this agent can use. The system will manage resource allocation dynamically.
